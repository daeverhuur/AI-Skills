## NotebookLM

AI-powered research and content generation tool at notebooklm.google.com. Upload sources, get AI-generated outputs.

**Source types**: PDFs, Google Docs, Google Slides, web URLs, YouTube videos, plain text, audio files

**Output formats**:

| Format | Description |
|--------|-------------|
| Audio Podcasts | Two AI hosts discuss your sources in conversational podcast format |
| Written Reports | Structured analysis with citations from your sources |
| Mind Maps | Visual topic relationship diagrams |
| Flashcards | Study cards with questions and answers from source material |
| Quizzes | Multiple choice and open-ended questions |
| Infographics | Visual data representations (powered by Nano Banana) |
| Slide Decks | Presentation-ready slides summarizing key points |
| Briefing Docs | Executive summary documents |
| Study Guides | Comprehensive learning materials |
| FAQs | Automatically generated Q&A from sources |
| Timelines | Chronological event visualization |

**Key features**:
- Up to 50 sources per notebook
- Each source up to 500,000 words
- Grounded responses (always cites specific sources, never hallucinates beyond sources)
- Interactive Q&A with sources
- Notebook sharing and collaboration
- Source-level citations with exact quotes

**Audio Overview features**:
- Two AI voices have natural conversation about your content
- Four formats (Sep 2025): Deep Dive, Brief, Critique, Debate
- Available in 80+ languages (Sep 2025)
- Interactive controls (Mar 2025): Smart Pause, Section Jump, Instant Replay, Summary Mode
- Customizable: tone, length, humor/formality, topic focus
- Download as MP3

**NotebookLM Plus** (Oct 2025): Enterprise version via Google Workspace and Google Cloud. Powered by Gemini 3 architecture with agentic research capabilities — can deploy Deep Research Agents to browse the live web for missing data.

**Video Overviews** (launched Jul 2025, 100% availability Aug 4, 2025):
- Narrated slides with AI-generated visuals, diagrams, quotes, and key numbers
- Nano Banana integration for image generation
- Six visual styles: Watercolor, Papercraft, Anime, Whiteboard, Retro Print, Heritage
- Two formats: Explainer (structured, comprehensive) and Brief (bite-sized)
- Available on Android and iOS (Jan 2026)
- Available to all ages including Workspace for Education (Oct 2025)

**Best uses**:
- Research paper analysis
- Meeting notes synthesis
- Course material preparation
- Content repurposing (turn documents into podcasts)
- Due diligence research
- Literature review
- Onboarding new team members on complex documentation

## Google Stitch

AI design tool at stitch.withgoogle.com for redesigning websites and apps from screenshots.

**How it works**:
1. Upload a screenshot of any website, app, or UI
2. Describe desired changes or just ask for a redesign
3. Stitch generates a new design maintaining the content but with improved layout, typography, colors
4. Export as code (HTML/CSS) or design files

**Key capabilities**:
- Screenshot-to-redesign: upload any UI, get an improved version
- Style transfer: apply one site's design language to another site's content
- Responsive design generation
- Component-level editing
- Export to code or design tools
- Powered by Gemini's visual understanding

**Use cases**:
- Rapid website redesign prototyping
- UI/UX improvement suggestions
- Landing page variant generation
- Design system exploration
- Competitive design analysis
- Quick mockup creation from existing sites

## Disco / GenTabs

Experimental standalone app (not a Chrome extension) that reimagines web browsing with AI. Announced Dec 11, 2025 by Google Labs. Powered by Gemini 3. Currently macOS-only, waitlist-based.

**How GenTabs works**:
1. Open multiple tabs with content you want to remix
2. Disco analyzes open tabs and chat history proactively
3. Generates interactive web applications (widgets) to help complete tasks — no code required
4. Describe what you need in natural language and refine iteratively

**Use cases**:
- Combine data from multiple sources into a dashboard
- Create comparison tools from product pages
- Build study guides from multiple articles
- Generate reports from scattered web sources
- Create custom views of aggregated web content
