# Advertising Audit Checklists

Cross-platform audit checklists for setup verification and performance optimization.

## Google Ads Audit Checklist

### Account Structure
- [ ] Manager account (MCC) set up for organization
- [ ] Campaigns organized by service/product area
- [ ] Ad groups themed by keyword intent (not too broad, not single-keyword)
- [ ] Naming conventions consistent and descriptive
- [ ] Account linked to Google Analytics 4
- [ ] Account linked to Google Merchant Center (if eCommerce)

### Conversion Tracking
- [ ] Google Ads conversion tag installed and firing
- [ ] Google Tag Manager set up (preferred method)
- [ ] All key conversions tracked (forms, calls, purchases)
- [ ] Enhanced conversions enabled
- [ ] Conversion values assigned correctly
- [ ] Attribution model set to data-driven
- [ ] Test conversions firing correctly in Tag Assistant

### Keywords
- [ ] Keywords relevant to ad groups they belong to
- [ ] Match types appropriate (phrase + exact, not just broad)
- [ ] No duplicate keywords across ad groups
- [ ] Search terms report reviewed (weekly)
- [ ] Negative keyword list comprehensive and maintained
- [ ] Shared negative keyword lists used where appropriate
- [ ] Low-performing keywords paused or removed

### Quality Score
- [ ] Average Quality Score 7+ for main keywords
- [ ] Keywords below 5 QS identified and action-planned
- [ ] Ad copy relevant to keywords in each ad group
- [ ] Landing pages match ad messaging
- [ ] Landing pages fast-loading and mobile-friendly

### Ads
- [ ] Responsive search ads in every ad group
- [ ] At least 8-10 unique headlines per RSA
- [ ] Ad strength "Good" or "Excellent" for all ads
- [ ] Headlines cover: keyword matching, benefits, CTAs
- [ ] Descriptions informative with clear value proposition
- [ ] Display URLs use path fields effectively
- [ ] A/B testing active (at least 2 RSAs per ad group)

### Extensions/Assets
- [ ] Sitelink extensions (4-6 minimum)
- [ ] Callout extensions (4+ highlights)
- [ ] Structured snippet extensions
- [ ] Call extensions (if phone leads relevant)
- [ ] Location extensions (if local business)
- [ ] Image extensions
- [ ] Price extensions (if applicable)
- [ ] Promotion extensions (for active offers)

### Bidding & Budget
- [ ] Bidding strategy appropriate for data volume
- [ ] Sufficient conversion data for smart bidding (30+/month)
- [ ] Daily budgets not limiting high-performing campaigns
- [ ] Budget allocation weighted toward best-performing campaigns
- [ ] Bid adjustments reviewed (device, location, time)

### Audiences
- [ ] Remarketing lists active and populated
- [ ] In-market and custom audiences applied (observation mode)
- [ ] Customer match lists uploaded (if available)
- [ ] Audience exclusions set (existing customers from acquisition campaigns)

## Facebook/Meta Ads Audit Checklist

### Account Setup
- [ ] Business Manager configured correctly
- [ ] Ad account created within Business Manager
- [ ] Team roles and permissions set appropriately
- [ ] Payment method active and billing current
- [ ] Business verification completed

### Tracking
- [ ] Facebook Pixel installed and active
- [ ] Pixel events firing correctly (test with Events Manager)
- [ ] Conversions API (CAPI) set up for server-side tracking
- [ ] Event deduplication configured (Pixel + CAPI)
- [ ] Domain verification completed
- [ ] Aggregated Event Measurement configured (for iOS 14+)
- [ ] Priority events ranked correctly (8 events max per domain)

### Campaign Structure
- [ ] Clear naming conventions (campaign/ad set/ad level)
- [ ] Campaign objectives match business goals
- [ ] CBO vs ABO chosen appropriately per campaign
- [ ] Not too many ad sets competing for same audience
- [ ] Testing campaigns separated from scaling campaigns

### Audiences
- [ ] Custom audiences created (website visitors, customer list, engagement)
- [ ] Lookalike audiences built from best sources (purchasers, high-value customers)
- [ ] Audience sizes appropriate (not too narrow, not too broad)
- [ ] Audience exclusions active (purchasers excluded from acquisition)
- [ ] Funnel structure: TOF/MOF/BOF audiences defined

### Creative
- [ ] Multiple ad formats tested (image, video, carousel)
- [ ] Ad copy has clear value proposition and CTA
- [ ] Primary text, headline, and description all populated
- [ ] Creative refreshed regularly (every 2-4 weeks for active campaigns)
- [ ] Video ads have captions/subtitles
- [ ] Aspect ratios optimized per placement

### Performance
- [ ] ROAS or CPA within target thresholds
- [ ] Frequency not too high (<3 for cold audiences, <7 for retargeting)
- [ ] CTR above 1% for cold traffic, 3%+ for retargeting
- [ ] No ad fatigue (declining CTR with rising frequency)
- [ ] Learning phase completed for active ad sets
- [ ] Attribution settings reviewed and understood

## Social Media Presence Audit

### Profile Optimization
- [ ] All platforms have complete, up-to-date profiles
- [ ] Consistent branding across platforms (logo, cover, colors)
- [ ] Bio/About includes clear value proposition and CTA
- [ ] Link in bio active and tracking (UTM parameters)
- [ ] Contact information accurate and complete
- [ ] Business category set correctly

### Content
- [ ] Posting frequency meets platform minimums
- [ ] Content mix balanced (educational, entertaining, promotional)
- [ ] Short-form video content present (Reels, TikTok, Shorts)
- [ ] Content calendar maintained and followed
- [ ] Visual quality consistent and on-brand
- [ ] Captions include CTAs and engagement prompts

### Engagement
- [ ] Comments responded to within 2 hours (business hours)
- [ ] DMs answered promptly
- [ ] Negative feedback handled professionally
- [ ] Community actively nurtured (not just broadcasting)
- [ ] UGC encouraged and featured

### Growth & Performance
- [ ] Follower growth rate positive and trending up
- [ ] Engagement rate above platform averages (2-5%)
- [ ] Best-performing content types identified
- [ ] Analytics reviewed weekly, strategy adjusted monthly
- [ ] Competitor benchmarking done quarterly

## Digital Marketing Strategy Audit

### Foundation
- [ ] Business goals clearly defined (SMART)
- [ ] Target audience documented with buyer personas
- [ ] Competitive analysis completed
- [ ] Marketing budget allocated with rationale
- [ ] KPIs defined for each channel

### Website
- [ ] Mobile-responsive design
- [ ] Page speed under 3 seconds (test with PageSpeed Insights)
- [ ] SSL certificate installed (HTTPS)
- [ ] Clear navigation and information architecture
- [ ] CTAs visible and compelling on every key page
- [ ] Contact information accessible from any page
- [ ] Service/product pages segmented and specific
- [ ] Trust elements present (reviews, testimonials, badges, logos)
- [ ] Forms optimized (minimal fields, clear labels)

### Analytics & Tracking
- [ ] Google Analytics 4 installed and configured
- [ ] UTM parameters used consistently across all campaigns
- [ ] Conversion goals defined and tracking
- [ ] Marketing attribution model selected
- [ ] Monthly reporting in place with actionable insights

### Multi-Channel Integration
- [ ] Consistent messaging across all channels
- [ ] Retargeting active across platforms
- [ ] Email nurture sequences for paid leads
- [ ] Content supports multiple channels (repurposed)
- [ ] Budget allocation reviewed monthly based on performance

### Email Marketing
- [ ] Email list growing (lead magnets, signup forms)
- [ ] Welcome sequence automated
- [ ] Regular email cadence established
- [ ] Segmentation in place
- [ ] Email authentication set up (SPF, DKIM, DMARC)
- [ ] Unsubscribe rate below 0.5%

## Quick-Start Setup Checklists

### Google Ads Quick Start
1. [ ] Create Manager Account (MCC)
2. [ ] Create Google Ads account under MCC
3. [ ] Install Google Tag Manager on website
4. [ ] Set up conversion tracking (forms, calls, purchases)
5. [ ] Link Google Analytics 4
6. [ ] Do keyword research (Keyword Planner)
7. [ ] Refine and group keywords into ad groups
8. [ ] Build negative keyword list
9. [ ] Write responsive search ads for each ad group
10. [ ] Set up all relevant ad extensions
11. [ ] Set budget and bidding strategy (start with Maximize Clicks)
12. [ ] Launch, monitor daily for first week

### Facebook Ads Quick Start
1. [ ] Set up Business Manager
2. [ ] Create ad account
3. [ ] Install Facebook Pixel via GTM
4. [ ] Set up Conversions API (CAPI)
5. [ ] Verify domain
6. [ ] Configure Aggregated Event Measurement
7. [ ] Create custom audiences (website visitors, customer list)
8. [ ] Build lookalike audiences
9. [ ] Create campaign with clear objective
10. [ ] Prepare creative (image + video + carousel variations)
11. [ ] Set up TOF campaign with 3-5 ad variations
12. [ ] Set up retargeting campaign (BOF)
13. [ ] Launch with $20-50/day budget
14. [ ] Monitor daily, optimize after 3-5 days of data

### Social Media Quick Start
1. [ ] Audit current presence across all platforms
2. [ ] Optimize profiles (bio, images, links)
3. [ ] Define content pillars (4-5 themes)
4. [ ] Create content calendar (2-4 weeks ahead)
5. [ ] Set up scheduling tool (Buffer, Hootsuite, Later)
6. [ ] Create first batch of content (10-15 posts)
7. [ ] Start posting consistently
8. [ ] Engage with audience daily (comments, DMs)
9. [ ] Track metrics weekly
10. [ ] Review and adjust monthly

### Full Digital Marketing Quick Start
1. [ ] Define goals and KPIs
2. [ ] Set up website analytics (GA4 + GTM)
3. [ ] Optimize website for conversions (CRO basics)
4. [ ] Set up email marketing (list building + welcome sequence)
5. [ ] Launch 1-2 paid channels (Google Ads + Facebook Ads)
6. [ ] Set up retargeting across platforms
7. [ ] Start content/social media strategy
8. [ ] Monitor and report weekly
9. [ ] Optimize monthly
10. [ ] Scale based on data after 2-3 months
